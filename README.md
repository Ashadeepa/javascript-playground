# javascript-playground

A simple hello world JavaScript repository.

## Getting Started

### Prerequisites
- Node.js installed on your system

### Running the Program

To run the hello world program, you can use either of the following methods:

1. Using npm:
   ```bash
   npm start
   ```

2. Using node directly:
   ```bash
   node hello.js
   ```

### Output
```
Hello, World!
```
